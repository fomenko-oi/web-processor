For run the project run command:
`make init`

_The command start docker containers, installs dependencies and builds assets._

Then, fill credentials for Yandex Music accounts in `.env` file.

`YANDEX_LOGIN` - login from yandex account that we are using for background parsing 

`YANDEX_COMMON_LOGIN` - login from yandex account that we are using for one the fly songs meta info parsing
